# 人性本善

> 人初性本善，天然无污染。  
> 道心自诚明，修真逆为仙。  
> 求真须务实，忠信炁自然。  
> 矛盾谋建设，当仁不让贤。  

```
刘意 2026年02月@天津
```

<details>
<summary style="line-height:4.0em;">
注释
</summary>
<article style="line-height:1.5em;text-indent:2em;">
诚者，常不欺人不自欺。以诚为本，可修真人。修诚者四：真，实，忠，信。真者，或真相真理，或真情真人。实者，或脚踏实地，或实事求是。忠者，取自曾子“为人谋而不忠乎”，乃尽心尽力。信者，取自曾子“与朋友交而不信乎”，乃真情实意。矛盾，乃人事困惑。建设，乃建其所设。建设者，每遇矛盾，当谋为所期之变因。仁者，合道之事也。当仁，不让于师，不让于贤，暗合“吾爱吾师，吾更爱真理”之意。
</article>
</details>

<summary style="line-height:4.0em;">
寓言
</summary>
<article style="line-height:1.5em;text-indent:2em;">
少年孤儿有三友，三友之父乃：精，亲，实。时值农忙，孤儿乞车运粮。求于精之子，精指桑骂槐曰：孤儿寡母，助之何益？求于亲之子，亲婉转其辞曰：吾见尤怜，非不助你，你母不知人情世故。求于实之子，或传实默而助之，或传实曰：善，贷之以农活何如？孤儿曰：然也。及其成年，孤儿复观之，其言皆人之常情，其行皆人之常理，并无不妥。然：精之子复精，亲之子复亲，实之子复实。精者，欲之直言，薄情寡义，众人远之。亲者，欲之婉言，虚情假意，众人虚之。孤儿仍友实。欲之直言乃善，薄情寡义非善；欲之婉言乃善，虚情假意非善。善者何也？心存柔软情存真切而已矣。
</article>
